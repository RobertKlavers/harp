CALIPSO_L2_CAL_LID_CLay

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of CALIPSO_L2_CAL_LID_CLay data.

field name type dimensions unit description
datetime_start double {time} [seconds since 2000-01-01] time at the start of the measurement
datetime double {time} [seconds since 2000-01-01] time during midpoint of the measurement
datetime_stop double {time} [seconds since 2000-01-01] time at the end of the measurement
latitude double {time} [degree_north] latitude of the ground pixel center
longitude double {time} [degree_east] longitude of the ground pixel center
tropopause_altitude double {time} [km] tropopause altitude
solar_zenith_angle double {time} [degree] solar zenith angle
solar_azimuth_angle double {time} [degree] solar azimuth angle
viewing_elevation_angle double {time} [degree] viewing elevation angle
scattering_angle double {time} [degree] scattering angle
altitude double {time, vertical} [km] altitude of the middle of the layer
altitude_bounds double {time, vertical, 2} [km] altitudes at the base and the top of the layer
pressure double {time, vertical} [hPa] pressure at the middle of the layer
pressure_bounds double {time, vertical, 2} [hPa] pressures at the base and the top of the layer
temperature double {time, vertical} [K] temperature at the middle of the layer
tropospheric_aerosol_optical_depth double {time} [] tropospheric aerosol optical depth
tropospheric_aerosol_optical_depth_uncertainty double {time} [] tropospheric aerosol optical depth uncertainty
stratospheric_aerosol_optical_depth double {time} [] stratospheric aerosol optical depth
stratospheric_aerosol_optical_depth_uncertainty double {time} [] stratospheric aerosol optical depth uncertainty
cloud_optical_depth double {time} [] cloud optical depth
cloud_optical_depth_uncertainty double {time} [] cloud optical depth uncertainty
index int32 {time}   zero-based index of the sample within the source product

Ingestion options

The table below lists the available ingestion options for CALIPSO_L2_CAL_LID_CLay products.

option name legal values description
wavelength 532, 1064 the wavelength whose measurements are ingested; option values are ‘532’ (default), ‘1064’

S5P_L2_O3_TCL

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of S5P_L2_O3_TCL data.

field name type dimensions unit description
datetime_start double {time} [seconds since 2000-01-01] coverage start time
datetime_stop double {time} [seconds since 2000-01-01] coverage stop time
latitude float {latitude} [degree_north] grid center latitudes
longitude float {longitude} [degree_east] grid center longitudes
tropospheric_O3_column_volume_mixing_ratio_dry_air float {time, latitude, longitude} [ppbv] tropospheric ozone mixing ratio
tropospheric_O3_column_volume_mixing_ratio_dry_air_uncertainty float {time, latitude, longitude} [ppbv] uncertainty of the tropospheric ozone mixing ratio
tropospheric_O3_column_volume_mixing_ratio_dry_air_validity int32 {time, latitude, longitude}   validity of the tropospheric ozone mixing ratio
tropospheric_O3_column_volume_mixing_ratio_dry_air_count int32 {time, latitude, longitude}   number of data used in the tropospheric ozone mixing ratio
tropospheric_O3_column_number_density float {time, latitude, longitude} [mol/m2] average tropospheric ozone column number density
tropospheric_O3_column_number_density_uncertainty float {time, latitude, longitude} [mol/m2] uncertainty of the average tropospheric ozone column number density
stratospheric_O3_column_number_density float {time, latitude, longitude} [mol/m2] average stratospheric ozone column number density
stratospheric_O3_column_number_density_uncertainty float {time, latitude, longitude} [mol/m2] uncertainty of the average stratospheric ozone column number density
O3_column_number_density float {time, latitude, longitude} [mol/m2] average total ozone column number density
O3_column_number_density_uncertainty float {time, latitude, longitude} [mol/m2] uncertainty of the average total ozone column number density
surface_albedo float {time, latitude, longitude} [] averaged surface albedo
surface_altitude float {time, latitude, longitude} [m] averaged surface height above mean sea level
surface_pressure float {time, latitude, longitude} [Pa] surface pressure
pressure_bounds float {time, latitude, longitude, 2} [Pa] pressure range of the retrieved ozone
index int32 {time}   zero-based index of the sample within the source product

Ingestion options

The table below lists the available ingestion options for S5P_L2_O3_TCL products.

option name legal values description
o3 ccd, csa whether to ingest the tropical tropospheric ozone column with the Convective Cloud Differential (CCD) method (o3=ccd, default) or with the Cloud Slicing Algorithm (CSA) (o3=csa)
o3_strat reference whether to ingest the ozone_stratospheric_vertical_column (default), or ozone_stratospheric_vertical_column_reference (o3_strat=reference); this option only applies if CCD data is ingested

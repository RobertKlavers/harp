ESACCI_OZONE_L3_TTOC

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of ESACCI_OZONE_L3_TTOC data.

field name type dimensions unit description
latitude double {latitude} [degree_north] latitude
longitude double {longitude} [degree_east] longitude
O3_column_number_density double {latitude, longitude} [DU] total ozone column number density
O3_column_number_density_uncertainty double {latitude, longitude} [DU] uncertainty of the total ozone column number density
stratospheric_O3_column_number_density double {latitude, longitude} [DU] stratospheric ozone column number density
stratospheric_O3_column_number_density_uncertainty double {latitude, longitude} [DU] uncertainty of the stratospheric ozone column number density
tropospheric_O3_column_number_density double {latitude, longitude} [DU] tropospheric ozone column number density
tropospheric_O3_column_number_density_uncertainty double {latitude, longitude} [DU] uncertainty of the tropospheric ozone column number density
tropospheric_O3_volume_mixing_ratio double {latitude, longitude} [ppbv] tropospheric ozone volume mixing ratio
tropospheric_O3_volume_mixing_ratio_uncertainty double {latitude, longitude} [ppbv] uncertainty of the tropospheric ozone volume mixing ratio
surface_albedo double {latitude, longitude} [] average surface area albedo
surface_altitude double {latitude, longitude} [m] surface altitude extracted from GTOPO30
cloud_albedo double {latitude, longitude} [] average cloud albedo
cloud_albedo_uncertainty double {latitude, longitude} [] cloud albedo uncertainty
cloud_top_height double {latitude, longitude} [km] altitude of the cloud top
cloud_top_height_uncertainty double {latitude, longitude} [km] uncertainty of altitude of the cloud top

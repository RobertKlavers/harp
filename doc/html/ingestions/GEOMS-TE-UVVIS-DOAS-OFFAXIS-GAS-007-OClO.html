GEOMS-TE-UVVIS-DOAS-OFFAXIS-GAS-007-OClO

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of GEOMS-TE-UVVIS-DOAS-OFFAXIS-GAS-007-OClO data.

field name type dimensions unit description
sensor_name string     name of the sensor
site_name string     name of the site at which the sensor is located
datetime double {time} [days since 2000-01-01] mean time of the measurement
datetime_start double {time} [days since 2000-01-01] start time of the measurement
datetime_stop double {time} [days since 2000-01-01] stop time of the measurement
sensor_latitude double   [degree_north] latitude of the sensor
sensor_longitude double   [degree_east] longitude of the sensor
sensor_altitude double   [m] altitude of the sensor relative to the location site
altitude double {time, vertical} [km] effective retrieval altitude
pressure double {time, vertical} [hPa] independent pressure profile
temperature double {time, vertical} [K] independent temperature profile
altitude_bounds double {time, vertical, 2} [km] lower and upper boundaries of the height layers
surface_wind_direction double {time} [degree] Wind direction at the station using WMO definition (wind from the north is 360; from the east is 90 and so on. No wind (calm) is 0)
surface_wind_speed double {time} [m/s] Wind speed at the station
solar_zenith_angle double {time} [degree] solar astronomical zenith angle
solar_azimuth_angle double {time} [degree] solar azimuth angle
viewing_azimuth_angle double {time} [degree] viewing azimuth angle of the sensor
viewing_zenith_angle double {time} [degree] viewing zenith angle of the sensor
latitude double {time, vertical} [degree_north] latitude of effective air mass at each altitude
longitude double {time, vertical} [degree_east] longitude of effective air mass at each altitude
cloud_type int8 {time}   cloud condition; enumeration values: clear_sky (0), thin_clouds (1), thick_clouds (2), broken_clouds (3)
tropospheric_aerosol_optical_depth double {time} [] tropospheric aerosol optical depth used for the retrieval
OClO_volume_mixing_ratio double {time, vertical} [ppmv] OClO volume mixing ratio
OClO_volume_mixing_ratio_covariance double {time, vertical, vertical} [(ppmv)2] covariance of the OClO volume mixing ratio
OClO_volume_mixing_ratio_uncertainty_random double {time, vertical} [ppmv] random uncertainty of the OClO volume mixing ratio
OClO_volume_mixing_ratio_uncertainty_systematic double {time, vertical} [ppmv] systematic uncertainty of the OClO volume mixing ratio
OClO_volume_mixing_ratio_apriori double {time, vertical} [ppmv] a priori OClO volume mixing ratio
OClO_volume_mixing_ratio_avk double {time, vertical, vertical} [] averaging kernel for the OClO volume mixing ratio
tropospheric_OClO_column_number_density double {time} [Pmolec cm-2] tropospheric OClO column number density
tropospheric_OClO_column_number_density_uncertainty_random double {time} [Pmolec cm-2] random uncertainty of the tropospheric OClO column number density
tropospheric_OClO_column_number_density_uncertainty_systematic double {time} [Pmolec cm-2] systematic uncertainty of the tropospheric OClO column number density
tropospheric_OClO_column_number_density_apriori double {time} [Pmolec cm-2] a priori tropospheric OClO column number density
tropospheric_OClO_column_number_density_avk double {time, vertical} [] averaging kernel for the tropospheric OClO column number density
OClO_column_number_density double {time, vertical} [Pmolec cm-2] OClO partial column number density profile
OClO_column_number_density_apriori double {time, vertical} [Pmolec cm-2] a priori OClO partial column number density profile
index int32 {time}   zero-based index of the sample within the source product

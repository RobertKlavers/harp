ESACCI_AEROSOL_L3_AATSR

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of ESACCI_AEROSOL_L3_AATSR data.

field name type dimensions unit description
latitude double {latitude} [degree_north] latitude of the ground pixel center
longitude double {longitude} [degree_east] longitude of the ground pixel center
aerosol_optical_depth double {spectral, latitude, longitude} [] aerosol optical depth
aerosol_optical_depth_uncertainty double {spectral, latitude, longitude} [] uncertainty of the aerosol optical depth
wavelength double {spectral} [nm] wavelengths of the measurements
sensor_zenith_angle double {latitude, longitude} [degree] sensor zenith angle
solar_zenith_angle double {latitude, longitude} [degree] solar zenith angle

Mapping description

The table below details where and how each variable was retrieved from the input product.

field name mapping description
latitude path /latitude[]
longitude path /longitude[]
aerosol_optical_depth path /AOD550_mean[], /AOD670_mean[], /AOD870_mean[], /AOD1600_mean[]
description depending on how the data is processed, data is not always available for all wavelengths. If the data is not available, NaN values are used.
aerosol_optical_depth_uncertainty path /AOD550_sdev[], /AOD670_sdev[], /AOD870_sdev[], /AOD1600_sdev[]
description depending on how the data is processed, uncertainty data is not always available for all wavelengths. If the data is not available, NaN values are used.
wavelength description fixed values
sensor_zenith_angle available optional
path /satellite_zenith_mean[]
solar_zenith_angle available optional
path /sun_zenith_mean[]

Ingestion options

The table below lists the available ingestion options for GOME_L1_EXTRACTED products.

option name

legal values

description

band

band-1a, band-1b, band-2a, band-2b, band-3, band-4, blind-1a, straylight-1a, straylight-1b, straylight-2a

only include data from the specified band (‘band-1a’, ‘band-1b’, ‘band-2a’, ‘band-2b’, ‘band-3’, ‘band-4’, ‘blind-1a’, ‘straylight-1a’, ‘straylight-1b’, ‘straylight-2a’); by default data from all bands is retrieved

data

sun_reference

retrieve the measured radiances (default) or the sun spectra (data=sun_reference)

This definition is only applicable when: data unset

The table below lists the variables that are present in the HARP product that results from an ingestion of GEOMS-TE-FTIR-TCCON-005 data.

HF_column_volume_mixing_ratio_dry_air float {time} [pptv] 0.2095 * column_HF/column_O2
HF_column_volume_mixing_ratio_dry_air_uncertainty float {time} [pptv] total random uncertainty on the retrieved total column (without smoothing error)

HF_column_number_density_avk float {time, vertical} [] averaging kernel matrix for the total HF vertical column

H2O_column_number_density_avk float {time, vertical} [] averaging kernel matrix for the total H2O vertical column
CO_column_volume_mixing_ratio_dry_air float {time} [ppbv] 0.2095 * column_CO/column_O2
